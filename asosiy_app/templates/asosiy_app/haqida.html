Loyiha haqida

Django Shablon Loyihasi

Bu loyiha Django freymvorki asosida yaratilgan to'liq funksional veb-ilova shablonidir. Loyiha barcha asosiy imkoniyatlar bilan jihozlangan va o'zbek tilida to'liq izohlangan.

Asosiy xususiyatlar:

  • To'liq o'zbekcha izohlar va dokumentatsiya
  • Tailwind CSS bilan zamonaviy dizayn
  • Foydalanuvchi autentifikatsiyasi (kirish, ro'yxatdan o'tish)
  • Profil boshqaruvi
  • CRUD operatsiyalari (Yaratish, O'qish, Yangilash, O'chirish)
  • Django Signals tizimi
  • Admin panel sozlamalari
  • Formalar va validatsiya
  • Ko'p ma'lumotlar bazasiga moslik (SQLite, PostgreSQL, MySQL)
  • Media va statik fayllar boshqaruvi

Texnologiyalar:

Python

Django

Tailwind CSS

SQL
